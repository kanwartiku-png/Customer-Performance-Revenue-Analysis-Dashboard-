# Customer Performance & Revenue Analysis Dashboard

## Project Overview

A Power BI dashboard designed to analyze customer behavior, revenue generation, profitability, and sales performance across multiple channels and regions.

## Business Objective

To provide actionable business insights through customer and revenue analytics.

## Tools Used

- Power BI
- SQL
- Excel
- DAX
- Power Query

## Key KPIs

- Net Sales
- Profit
- Profit Margin
- Customer Count
- Revenue by State
- Revenue by Category
- Revenue by Channel

## Dashboard Features

- Customer Segmentation
- Revenue Trend Analysis
- Profitability Analysis
- State Performance Dashboard
- Sales Channel Analysis

## Key Insights

- Home & Kitchen generated highest revenue.
- Amazon Prime was the top-performing sales channel.
- B2C customers contributed the largest share of sales.
- Haryana generated the highest revenue among all states.

## Dataset

1,500 customer transaction records.
